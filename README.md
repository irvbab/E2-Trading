# 📈 Analyse de Marché et Prédiction (Blocs E2 & E5)

## 📝 Présentation du Projet
Ce projet analyse les corrélations entre les cryptomonnaies (BTC) et les actions technologiques (AMZN, NFLX, etc.) pour prédire les prix futurs via un modèle de **Régression Linéaire**.

---

## 🛠️ Compétences Validées

### **Bloc E2 : Mathématiques & Statistiques**
*   **Analyse de corrélation** : Étude des relations entre actifs (Heatmap).
*   **Indicateurs financiers** : Calcul du Ratio de Sharpe et des Moyennes Mobiles.
*   **Machine Learning** : Modélisation prédictive avec Scikit-Learn.
*   **Évaluation** : Mesure de la précision via MAE et score R².

### **Bloc E5 : Développement et Déploiement**
*   **Gestion de version** : Utilisation de Git et GitHub pour le suivi du code.
*   **Documentation** : README détaillé et fichier `requirements.txt` pour la portabilité.
